# Phishing Email Detection Model using Machine Learning

A standalone, lightweight text classification engine built to detect malicious phishing attempts in text emails using a custom-implemented Naive Bayes Classifier algorithm optimized for resource-constrained environments.

## 🚀 Key Project Features
- **Zero Heavy Dependencies**: Built from scratch using native mathematical arrays, bypassing bulky library dependencies.
- **Laplace Smoothing Mechanics**: Includes uniform token probability corrections to process unseen vocabulary without crash conditions.
- **Embedded Evaluation Suite**: Dynamically assesses performance metrics upon every program execution.
